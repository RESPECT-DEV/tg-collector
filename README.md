# tg-collector

Телеграм-бот для сбора сообщений из чатов и отправки их на бэкенд.

## Установка

```bash
npm install
cp .env.example .env
# заполнить .env своими значениями
```

## Переменные окружения

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `WEBHOOK_URL` | Публичный HTTPS-адрес сервера (без слэша в конце) |
| `BACKEND_ENDPOINT` | URL эндпоинта, куда бот шлёт сообщения |
| `PORT` | Порт Express-сервера (по умолчанию 3000) |

## Запуск

```bash
node index.js
```

## Структура

```
index.js      — Express-сервер, приём webhook от Telegram
bot.js        — инициализация бота, регистрация webhook
handlers.js   — парсинг сообщения, отправка на бэкенд
```
